# KandR

Implementation of the exercises in K&amp;R C book

```
Usage:
  make <ex>        Build src/<ex>.c -> build/<ex>     (e.g. make 1-1)
  make run <ex>    Build (if needed) and run          (e.g. make run 1-1)
  make clean       Remove the build directory

Layout:
  src/1-1.c, src/2-4.c, ...
```
